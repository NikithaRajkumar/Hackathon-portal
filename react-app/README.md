# Virtual Hackathon Platform - React App

## Complete Setup & Run

### 1. Install Dependencies
```bash
cd "c:\Users\nikit\OneDrive\Desktop\hackthon portal\react-app"
npm install
```

### 2. Start Development Server
```bash
npm start
```

The app will automatically open at `http://localhost:3000`

### 3. Build for Production
```bash
npm run build
```
